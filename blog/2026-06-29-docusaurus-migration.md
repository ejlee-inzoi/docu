ModKit 문서가 기존 MkDocs에서 **Docusaurus**로 이전되었습니다. 더 빠르고,
검색이 편하며, 다크 모드를 지원합니다.

<!-- truncate -->

## 무엇이 좋아졌나요?

| 항목 | 이전(MkDocs) | 현재(Docusaurus) |
| --- | --- | --- |
| 테마 | Material | React 기반 커스텀 |
| 검색 | 플러그인 | 내장 + Algolia 옵션 |
| 다크 모드 | 토글 | 토글 + 시스템 설정 연동 |
| 확장 | Python 플러그인 | React/MDX 컴포넌트 |
