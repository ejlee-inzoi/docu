# 01. Overview

The Skin Modkit is a tool provided to customize or replace the Head Skin and Body Skin of inZOI characters.
Skin is the most fundamental layer that composes a character’s appearance, and the Skin Modkit allows creators to replace the Color Texture for both the face and body.

The Skin Modkit is composed of the following elements:

---

**Head Materials / Body Material**

- Head Materials define the Material Instances used to render the character’s face.  
- Each Material Instance is linked with the default facial textures used in inZOI, and users can replace them with new Color Textures through the **Head Texture** field in the Texture section.
- Body Material defines the Material Instance used to render the entire body.  
- Each Material Instance is linked with the default body textures used in inZOI, and users can replace them with new Color Textures through the **Body Texture** field.

---

**Texture Input (Head / Body / Thumbnail)**

- In the Texture section, users can import Head and Body Color Textures to replace the existing skin textures.  
- Additionally, a Thumbnail image can be assigned, which will appear in the skin selection UI.

---

**Skin Properties**

- Skin Properties define the gender (Male/Female) and age group (Adult/Youth/Baby) that the skin applies to, ensuring the skin is correctly matched with the character data.  
- This allows skins created through the Modkit to be properly applied across different gender and age variations in the game.
