# 03. Skin Properties

The Skin Properties section defines how the created skin is matched to different character types.
The skin must be correctly aligned with basic character attributes such as gender and age group to ensure proper in-game application.
This information must be managed accurately to maintain compatibility with character data.

The Skin Properties section consists of the following items:

---

**Matching Body Age Id**

This field defines which Skin ID is matched to each age group.
The value is referenced when the character’s age group is changed in the customization system, and it also influences genetics in the game.
Different Body Skin IDs can be assigned for each age group such as Adult, Youth, and Baby.

---

**Matching Gender Id**

This field defines which Skin ID is matched when the character’s gender is changed.
The assigned ID is referenced during gender switching in the customization system, ensuring that the skin transitions correctly between male and female variants.

---

**Gender Type**

This field specifies the gender for which the current skin is created.

---

**Body Age Type**

This field specifies the age group for which the current skin is created.
