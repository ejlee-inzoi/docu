# DCC Tools

## Blender

1. Move the file to:  
   `\inZOIModKit\Extras\Blender`  
   **Do not unzip** the file.

2. In Blender, go to **Edit → Preferences → Add-ons**.

3. Click the **downward arrow icon** in the upper-right corner and select **‘Install From Disk...’**.

4. Select the following file:  
   `\inZOIModKit\Extras\BlenderPlugin\Blender.zip`

5. Once the add-on is successfully added,  
   you will see the **inZOI tab** appear under the **Tool Tab** on the right.

---

## Maya

1. Move the files to:  
   `\inZOIModKit\Extras\Maya`

2. In the `inZOI_ModKit_intoZOI` folder,  
   drag and drop the script `Maya_intoZOI_Install.py` into the Maya viewport.

3. An **inZOI icon** will be added to the **Shelf bar**, completing the installation.
