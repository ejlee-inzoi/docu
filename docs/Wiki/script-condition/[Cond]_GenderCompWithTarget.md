# [Cond]_GenderCompWithTarget

Overview
==

* Command to check whether the gender of the Self character is the same as or different from that of the Target character.

Parameters
==========

| Params      | Description                |
|-------------|----------------------------|
| BaseObject  | Not used                   |
| Command     | GenderCompWithTarget       |
| CompareType | Type of comparison         |

Examples
========

| Example                                 | BaseObject | Command              | CompareType | S1 | S2 | F1 | F2 | Prob |
|-----------------------------------------|------------|-----------------------|-------------|----|----|----|----|------|
| If my gender is the same as the target's    |            | GenderCompWithTarget | Equal       |    |    |    |    |      |
| If my gender is different from the target's |            | GenderCompWithTarget | NotEqual    |    |    |    |    |      |
