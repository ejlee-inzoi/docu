# [Cond]_IsForceableSale

Overview
==

* Checks whether the target object is subject to forfeiture (i.e., can be forcibly sold).

Parameter
=========

|         | Description                              |
|---------|------------------------------------------|
| BaseObject | The object to check the condition for |
| Command    | IsForceableSale                       |
| CompareType| Greater                               |

Examples
========

| Description                                 | BaseObject | Command          | CompareType |   |   |   |   |   |
|---------------------------------------------|------------|------------------|-------------|---|---|---|---|---|
| Check if the target object can be sold      | Target     | IsForceableSale  | Greater     |   |   |   |   |   |
