# [Cond]_IsObjectAuthor

Overview
==

* Checks whether the object was created by the current character (i.e., verifies authorship of the object).

Parameter
=========

| Params      | Description                              |
|-------------|------------------------------------------|
| Command     | IsObjectAuthor                           |
| CompareType | Comparison type (Equal / NotEqual)       |

Examples
========

| Example Description                     | BaseObject | Command         | CompareType | S1 | S2 | F1 | F2 | Prob |
|-----------------------------------------|------------|------------------|-------------|----|----|----|----|-------|
| If I am the author of the object        | Target     | IsObjectAuthor   | Equal       |    |    |    |    |       |
