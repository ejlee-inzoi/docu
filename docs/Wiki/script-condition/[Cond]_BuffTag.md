# [Cond]_BuffTag

Overview
==

- Checks whether a buff with a specific tag is present.

Parameters
==========

| Params      | Description |
|------------|-------------|
| BaseObject | Target object to check the condition |
| Command    | BuffTag |
| CompareType | Type of comparison operation |
| S1         | Buff tag name |

Examples
========

| Example | BaseObject | Command | CompareType | S1    | S2 | F1 | F2 | Prob |
|--------|------------|---------|-------------|--------|----|----|----|------|
| Check if the "Dream" BuffTag is present | Self | BuffTag | Equal | Dream |    |    |    |      |
