# [Cond]_HasSalesRights

Overview
==

* Checks whether the character has the rights to sell or cancel the sale of the target object.

Parameter
=========

| Parameter   | Description                             |
|-------------|-----------------------------------------|
| BaseObject  | The object to check the condition for   |
| Command     | HasSalesRights                          |
|             |                                         |

Examples
========

| Dialogue Text                             | [Condition] |         |         |         |
|-------------------------------------------|-------------|---------|---------|---------|
| BaseObject                                | Command     | S1      | S2      |
| Do I have the right to sell/cancel target?| Target      | HasSalesRights |         |         |
