# [Cond]_HasOwnedVehicle

Overview
==

* Condition to check whether a character entity owns a vehicle.

Parameter
=========

| Params      | Description                                               |
|-------------|-----------------------------------------------------------|
| BaseObject  | The character (Self/Target) whose vehicle ownership is being checked |
| Command     | HasOwnedVehicle                                           |
| CompareType | Comparison type                                           |

Examples
========

| **Example**                  | [Condition]        |         |         |
|-----------------------------|--------------------|---------|---------|
| BaseObject                  | Command            | CompareType |
| Do I own a vehicle?         | Self               | HasOwnedVehicle | Equal |
