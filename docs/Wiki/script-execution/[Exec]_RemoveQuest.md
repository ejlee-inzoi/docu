# [Exec]_RemoveQuest

Overview
==

* Removes a specific quest from a character.

Details
==

| Parameter   | Description                   |
|-------------|-------------------------------|
| BaseObject  | Object to execute the command |
| Command     | RemoveQuest                   |
| S1          | QuestId                       |

Examples
==

| Example                     | BaseObject | Command     | S1      | S2 | F1 | F2 | Prob |
|-----------------------------|------------|-------------|---------|----|----|----|------|
| Remove an assigned quest    | Self       | RemoveQuest | JobLess |    |    |    |      |
