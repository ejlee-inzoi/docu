# [Exec]_PutDownDetachCharacter

Overview
==

* Command to detach and put down the Joy held in the right hand of the base object.

Details
==

| Parameter   | Description                                  |
|-------------|----------------------------------------------|
| BaseObject  | Base object to execute put down (Self, Target) |
| Command     | PutDownDetachCharacter                       |

Examples
==

| Example                                              | BaseObject | Command              | S1 | S2 | F1 | F2 | Prob |
|------------------------------------------------------|------------|----------------------|----|----|----|----|------|
| Detach and put down the character in self's right hand  | Self       | PutDownDetachCharacter |    |    |    |    |      |
| Detach and put down the character in target's right hand| Target     | PutDownDetachCharacter |    |    |    |    |      |
