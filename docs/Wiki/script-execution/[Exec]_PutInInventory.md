# [Exec]_PutInInventory

Overview
==

* Puts the specified target object into the inventory.

Details
==

| Parameter   | Description                                              |
|-------------|----------------------------------------------------------|
| BaseObject  | Refers to the target object executed by the target character |
| Command     | PutInInventory                                           |
| S1          | -                                                        |
| S2          | -                                                        |
