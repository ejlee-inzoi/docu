# [Exec]_PickUpServedFoodObject

Overview
==

* The Self Joy picks up one slice (ServedFood) from the Target food object (PreparedFood) placed on the site and attaches it to their right hand.  
* This function both spawns and attaches the slice to the character’s hand.

Details
==

| --- | --- |
| Command | PickUpServedFoodObject |
