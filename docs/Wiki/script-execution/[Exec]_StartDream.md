# [Exec]_StartDream

Overview
==

* Starts a dream for the specified character.

Details
==

| Parameter   | Description                   |
|-------------|-------------------------------|
| BaseObject  | Object to execute the command |
| Command     | StartDream                    |

Examples
==

| Example                      | BaseObject | Command    |
|------------------------------|------------|------------|
| Start dream for self         | Self       | StartDream |
| Start dream for target       | Target     | StartDream |
