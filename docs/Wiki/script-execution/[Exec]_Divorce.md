# [Exec]_Divorce

Overview
==

* Executes a divorce script.  
* Converts the "MarriagePartner" relationship into a normal relationship.  
* Resets relationship stats as follows:  
  + Romance: -100  
  + Friendship: 0  
  + Business: 0

Details
==

| Parameter | Description |
| --- | --- |
| Command | Divorce |
