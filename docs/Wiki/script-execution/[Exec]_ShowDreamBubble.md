# [Exec]_ShowDreamBubble

Overview
==

* Displays a speech bubble based on the dream buff currently held by the target object (character).

Details
==

| Parameter   | Description                   |
|-------------|-------------------------------|
| BaseObject  | Character to execute the command |
| Command     | ShowDreamBubble               |

Examples
==

| Example                                         | BaseObject | Command          | S1 | S2 | F1 | F2 | Prob |
|-------------------------------------------------|------------|------------------|----|----|----|----|------|
| Show a dream-related speech bubble on self      | Self       | ShowDreamBubble  |    |    |    |    | 1.0  |
| Show a dream-related speech bubble on target    | Target     | ShowDreamBubble  |    |    |    |    | 1.0  |
