# [Exec]_DestroyCraftObject

Overview
==

* Removes a crafting GameObject that the character is currently in the process of creating from the world.

Details
==

| Parameter | Description |
| --- | --- |
| BaseObject | The object (usually the character) on which to execute the command |
| Command | DestroyCraftObject |

Example
==

| Example | BaseObject | Command | S1 | S2 | F1 | F2 | Prob |
| --- | --- | --- | --- | --- | --- | --- | --- |
| Discard the item currently being crafted through interaction | Self | DestroyCraftObject |  |  |  |  |  |
