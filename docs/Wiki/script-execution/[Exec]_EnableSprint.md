# [Exec]_EnableSprint

Overview
==

* Enables sprinting for the character.  
  + Used to lift a previously applied sprint restriction.  
* Note: Enabling sprint does **not** force the character to sprint — it only allows sprinting (e.g., when the Shift key is held or during zone graph movement).

Details
==

| Parameter | Description |
| --- | --- |
| BaseObject | The object on which to execute the command |
| Command | EnableSprint |

Example
==

| Example | BaseObject | Command | S1 | S2 | F1 | F2 | Prob |
| --- | --- | --- | --- | --- | --- | --- | --- |
| Enable sprinting for myself | Self | EnableSprint |  |  |  |  |  |
