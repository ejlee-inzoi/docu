# [Exec]_DetachGrabObject

Overview
==

* Puts down (destroys) the object currently being held.

Details
==

| Parameter | Description |
| --- | --- |
| BaseObject | The object on which to execute the command |
| Command | DetachGrabObject |
| S1 | - |

Example
==

| Example | BaseObject | Command | S1 | S2 | F1 | F2 | Prob |
| --- | --- | --- | --- | --- | --- | --- | --- |
| Detach (drop) the currently held object | Self | DetachGrabObject |  |  |  |  |  |
