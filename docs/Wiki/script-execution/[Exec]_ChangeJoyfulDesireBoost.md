# [Exec]_ChangeJoyfulDesireBoost

Overview
==

* Changes the boost applied to the Joyful (fun/enjoyment) desire.

Details
==

| Parameter | Description |
| --- | --- |
| BaseObject | The object on which to execute the command |
| Command | ChangeJoyfulDesireBoost |
| S1 | Buff ID to apply as the boost |
