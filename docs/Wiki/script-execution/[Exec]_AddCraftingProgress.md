# [Exec]_AddCraftingProgress

Overview
==

* Increases the progress of an item the character is currently crafting.

Details
==

| Parameter | Description |
| --- | --- |
| BaseObject | The object on which to execute the command |
| Command | AddCraftingProgress |
| F1 | Amount of progress to add |

Example
==

| Example | BaseObject | Command | S1 | S2 | F1 | F2 | Prob |
| --- | --- | --- | --- | --- | --- | --- | --- |
| Increase composition progress through crafting | Self | AddCraftingProgress |  |  | 1 |  |  |
