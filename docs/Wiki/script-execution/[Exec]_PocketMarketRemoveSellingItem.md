# [Exec]_PocketMarketRemoveSellingItem

Overview
==

* Removes an object currently listed for sale in the Pocket Market.

Details
==

| Parameter | Description |
| --- | --- |
| BaseObject | The GameObject to remove (Self or Target) |
| Command | PocketMarketRemoveSellingItem |

Example
==

| Example | BaseObject | Command | S1 | S2 | F1 | F2 | Prob |
| --- | --- | --- | --- | --- | --- | --- | --- |
| Remove the Target GameObject from the Pocket Market | Target | PocketMarketRemoveSellingItem |  |  |  |  |  |
| Remove the Self GameObject from the Pocket Market | Self | PocketMarketRemoveSellingItem |  |  |  |  |  |
