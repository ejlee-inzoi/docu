# [Exec]_PayForObjectExchange

Overview
==

* Deducts the cost for repairing or exchanging an object.

Details
==

| Parameter | Description |
| --- | --- |
| BaseObject | The object on which to execute the command |
| Command | PayForObjectExchange |
| S1 | Currency Data Table ID (=CurrencyType) |

Example
==

| Example | BaseObject | Command | S1 | S2 | F1 | F2 | Prob |
| --- | --- | --- | --- | --- | --- | --- | --- |
| Pay the repair fee for a broken refrigerator | Self | PayForObjectExchange | Mew |  |  |  |  |
