# [Exec]_UnfreezeVehicle

Overview
==

* Removes the flag that disables manual control of a vehicle.

Parameters
==========

| Parameter   | Description                   |
|-------------|-------------------------------|
| BaseObject  | Target                        |
| Command     | UnfreezeVehicle               |
| S1          | None                          |
| S2          | None                          |

|------------------------------------|------------|------------------|------|------|----|----|------|
| Remove manual control restriction  | Target     | UnfreezeVehicle  | None | None |    |    |      |
