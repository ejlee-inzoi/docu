# [Exec]_AddJobExp

Overview
==

* Increases the experience points (EXP) of the target's current job.

Details
==

| Parameter | Description |
| --- | --- |
| BaseObject | The object on which to execute the command |
| Command | AddJobExp |
| S1 | - |
| S2 | - |
| F1 | Amount of experience to add |
| F2 | - |

Example
==

| Example | BaseObject | Command | S1 | S2 | F1 | F2 | Prob |
| --- | --- | --- | --- | --- | --- | --- | --- |
| Increase the target's current job EXP by 5 | Self | AddJobExp |  |  | 5 |  | 1.0 |
