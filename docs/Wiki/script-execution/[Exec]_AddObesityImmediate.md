# [Exec]_AddObesityImmediate

Overview
==

* Adjusts obesity level and applies the change to appearance immediately.

Details
==

| Parameter | Description |
| --- | --- |
| BaseObject | The target to apply the obesity level change to |
| Command | AddObesityImmediate |
| S1 |  |
| F1 | Amount of obesity to add or remove (integer) |

Example
==

| Example | BaseObject | Command | S1 | S2 | F1 | F2 | Prob |
| --- | --- | --- | --- | --- | --- | --- | --- |
| Increase obesity by 50 for Self | Self | AddObesityImmediate |  |  | 50 |  |  |
| Decrease obesity by 50 for Self | Self | AddObesityImmediate |  |  | -50 |  |  |
