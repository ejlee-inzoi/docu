# [Exec]_ChildBirth

Overview
==

* Immediately triggers childbirth if the specified Joy is currently pregnant.

Details
==

| Parameter | Description |
| --- | --- |
| BaseObject | The Joy (character) who will give birth (Self or Target) |
| Command | ChildBirth |

Example
==

| Example | BaseObject | Command | S1 | S2 | F1 | F2 | Prob |
| --- | --- | --- | --- | --- | --- | --- | --- |
| Trigger childbirth for myself | Self | ChildBirth |  |  |  |  |  |
| Trigger childbirth for the target Joy | Target | ChildBirth |  |  |  |  |  |
