# [Exec]_AddSiteAccessApprove / RemoveSiteAccessApprove

Overview
==

* Grants or removes permission for the specified object to use the facilities at the site where the target is currently located.  
  (e.g., when visiting another family’s home)

Details
==

| Parameter | Description |
| --- | --- |
| BaseObject | The object on which to execute the command |
| Command | AddSiteAccessApprove / RemoveSiteAccessApprove |

Example
==

| Example | BaseObject | Command | S1 | S2 | F1 | F2 | Prob |
| --- | --- | --- | --- | --- | --- | --- | --- |
| Grant site access permission to myself | Self | AddSiteAccessApprove |  |  |  |  |  |
| Revoke site access permission from myself | Self | RemoveSiteAccessApprove |  |  |  |  |  |
