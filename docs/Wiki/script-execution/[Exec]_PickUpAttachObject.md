# [Exec]_PickUpAttachObject

Overview
==

* Attaches a Target Carriable GameObject (already spawned) to the right hand of the Self character.  
* Will not execute if the target GameObject is not carriable.

Details
==

| Parameter | Description |
| --- | --- |
| Command | PickUpAttachObject |

Example
==

| Example | BaseObject | Command | S1 | S2 | F1 | F2 | Prob |
| --- | --- | --- | --- | --- | --- | --- | --- |
| Attach a Carriable GameObject to my right hand | Self | PickUpAttachObject |  |  |  |  |  |
