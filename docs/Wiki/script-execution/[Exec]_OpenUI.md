# [Exec]_OpenUI

Overview
==

* Opens a specified UI.  
  + Note: The `BaseObject` must be the currently possessed Joy for the script to work.

Details
==

| Parameter | Description |
| --- | --- |
| BaseObject | The object on which to execute the command |
| Command | OpenUI |
| S1 | UI type (EB1UIType) |

Example
==

| Example | BaseObject | Command | S1 | S2 | F1 | F2 | Prob |
| --- | --- | --- | --- | --- | --- | --- | --- |
| Open the Inventory UI | Self | OpenUI | Inventory |  |  |  |  |
