# [Exec]_AddMuscleImmediate

Overview
==

* Adjusts muscle mass and applies the change to appearance immediately.

Details
==

| Parameter | Description |
| --- | --- |
| BaseObject | The target to apply the muscle mass change to |
| Command | AddMuscleImmediate |
| S1 |  |
| F1 | Amount of muscle mass to add or remove (integer) |

Example
==

| Example | BaseObject | Command | S1 | S2 | F1 | F2 | Prob |
| --- | --- | --- | --- | --- | --- | --- | --- |
| Increase muscle mass by 50 for Self | Self | AddMuscleImmediate |  |  | 50 |  |  |
| Decrease muscle mass by 50 for Self | Self | AddMuscleImmediate |  |  | -50 |  |  |
