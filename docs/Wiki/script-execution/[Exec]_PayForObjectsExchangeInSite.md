# [Exec]_PayForObjectsExchangeInSite

Overview
==

* Deducts the cost for repairing or replacing all damaged objects within the site.

Details
==

| Parameter | Description |
| --- | --- |
| BaseObject | The object on which to execute the command |
| Command | PayForObjectsExchangeInSite |
| S1 | Currency Data Table ID (=CurrencyType) |

Example
==

| Example | BaseObject | Command | S1 | S2 | F1 | F2 | Prob |
| --- | --- | --- | --- | --- | --- | --- | --- |
| Pay for replacing all burnt furniture within the site | Self | PayForObjectsExchangeInSite | Mew |  |  |  |  |
