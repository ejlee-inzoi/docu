# [Exec]_AddRandomBuff

Overview
==

* Randomly adds one Buff from a specified pool.

Details
==

| Parameter | Description |
| --- | --- |
| BaseObject | The object on which to execute the command |
| Command | AddRandomBuff |
| S1 | Random Buff Pool Table ID (from RandomBuffPool.json) |

Example
==

| Example | BaseObject | Command | S1 | S2 | F1 | F2 | Prob |
| --- | --- | --- | --- | --- | --- | --- | --- |
| Add a random Buff from the RandomBuffTest pool to myself | Self | AddRandomBuff | RandomBuffTest |  |  |  |  |
