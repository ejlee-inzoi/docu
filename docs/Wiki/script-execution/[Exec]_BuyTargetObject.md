# [Exec]_BuyTargetObject

Overview
==

* Pays the price for the target object.  
* Once payment is made, the object is no longer marked for sale.  
* Note: Ownership does **not** actually transfer to the buyer.

Details
==

| Parameter | Description |
| --- | --- |
| BaseObject | The object on which to execute the command |
| Command | BuyTargetObject |
