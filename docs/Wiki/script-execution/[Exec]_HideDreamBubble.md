# [Exec]_HideDreamBubble

Overview
==

* Hides the speech/dream bubble that is currently displayed above the specified character.

Details
==

| **Parameter** | **Description** |
| --- | --- |
| BaseObject | The object (character) on which to execute the command |
| Command | HideDreamBubble |

Example
==

| **Example** | **BaseObject** | **Command** | **S1** | **S2** | **F1** | **F2** | **Prob** |
| --- | --- | --- | --- | --- | --- | --- | --- |
| Hide the dream bubble shown on myself | Self | HideDreamBubble |  |  |  |  | 1.0 |
| Hide the dream bubble shown on the target | Target | HideDreamBubble |  |  |  |  | 1.0 |
