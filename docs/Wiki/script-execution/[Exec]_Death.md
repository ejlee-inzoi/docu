# [Exec]_Death

Overview
==

* Instantly causes the specified character to die.

Details
==

| Parameter | Description |
| --- | --- |
| BaseObject | The object (character) to execute the command on |
| Command | Death |

Example
==

| Example | BaseObject | Command | S1 | S2 | F1 | F2 | Prob |
| --- | --- | --- | --- | --- | --- | --- | --- |
| Cause my character to die | Self | Death |  |  |  |  |  |
| Cause the target character to die | Target | Death |  |  |  |  |  |
