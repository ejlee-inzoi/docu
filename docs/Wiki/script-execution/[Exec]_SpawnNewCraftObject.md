# [Exec]_SpawnNewCraftObject

Overview
==

* Spawns a Crafting GameObject, created directly by the character, into the world.

Details
==

| Parameter   | Description                   |
|-------------|-------------------------------|
| BaseObject  | Object to execute the command |
| Command     | SpawnNewCraftObject           |

Examples
==

| Example                                                          | BaseObject | Command             | S1 | S2 | F1 | F2 | Prob |
|------------------------------------------------------------------|------------|----------------------|----|----|----|----|------|
| Spawn the currently crafting item via interaction               | Self       | SpawnNewCraftObject  |    |    |    |    |      |
