# [Exec]_OpenGiftBox

Overview
==

* Deletes the gift box and spawns the contained gift object into the character’s hand.

Details
==

| Parameter | Description |
| --- | --- |
| BaseObject | The object on which to execute the command (**Target only**) |
| Command | OpenGiftBox |

Example
==

| Example | BaseObject | Command | S1 | S2 | F1 | F2 | Prob |
| --- | --- | --- | --- | --- | --- | --- | --- |
| Open the target (gift box) | Target | OpenGiftBox |  |  |  |  |  |
