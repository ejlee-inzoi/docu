# [Exec]_UnrequitedLove

Overview
==

* Sets up a one-sided (unrequited) love relationship with a given probability.

Details
==

| Parameter   | Description                                                                      |
|-------------|----------------------------------------------------------------------------------|
| BaseObject  | Target of the command:  
                 - `Self`: I fall in love with the other  
                 - `Target`: The other falls in love with me |
| Command     | UnrequitedLove                                                                   |
| F1          | Probability of unrequited love (0.0 ~ 1.0)                                       |

Examples
==

| Execution Condition                | BaseObject | Command        | F1 |
|-----------------------------------|------------|----------------|----|
| After verifying conditions for love | Target     | UnrequitedLove | 1  |
