# [Exec]_PickUpAttachCharacter

Overview
==

* Attaches the Target character to the right hand of the Self character.  
* Will not execute if the right hand is already holding something.

Details
==

| Parameter | Description |
| --- | --- |
| Command | PickUpAttachCharacter |

==

| --- | --- | --- | --- | --- | --- | --- | --- |
