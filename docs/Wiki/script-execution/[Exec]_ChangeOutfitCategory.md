# [Exec]_ChangeOutfitCategory

Overview
==

* Changes the character’s outfit based on the specified OutfitCategory.

Available OutfitCategories:  
Everyday, Formal, School, Sleep, Party, Sports, Swimwear, Summer, Winter

Details
==

| Parameter | Description |
| --- | --- |
| BaseObject | The object on which to execute the command |
| Command | ChangeOutfitCategory |
| S1 | OutfitCategory value |

Example
==

| Example | BaseObject | Command | S1 | S2 | F1 | F2 | Prob |
| --- | --- | --- | --- | --- | --- | --- | --- |
| Change outfit to Sleepwear | Self | ChangeOutfitCategory | Sleep |  |  |  |  |
| Change outfit to Partywear | Self | ChangeOutfitCategory | Party |  |  |  |  |
