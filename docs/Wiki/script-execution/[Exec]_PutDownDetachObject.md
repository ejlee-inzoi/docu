# [Exec]_PutDownDetachObject

Overview
==

* Function to put down a carriable GameObject held in the right hand of self Joy without despawning it, onto something (target object, ground, or bag).

Details
==

| Parameter | Description         |
|-----------|---------------------|
| Command   | PutDownDetachObject |

Examples
==

| Example                              | BaseObject | Command            | S1 | S2 | F1 | F2 | Prob |
|--------------------------------------|------------|--------------------|----|----|----|----|------|
| Put down the object in my right hand | Self       | PutDownDetachObject |    |    |    |    |      |
