# stat

Overview
==

- 6 commands in this category.

Commands
==

| Command | Name | Description |
|---|---|---|
| [`stat.add`](stat.add.md) | Add Stat | Adds the given value to a stat on the target character. |
| [`stat.freeze`](stat.freeze.md) | Freeze Stats | Freezes (enabled=true) or unfreezes stat tick for the target character. |
| [`stat.get_all`](stat.get_all.md) | Get All Stats | Returns all stats with value, min, max, pct, and noDecay for the target character. |
| [`stat.set`](stat.set.md) | Set Stat | Sets a stat to the given value on the target character. |
| [`stat.set_all`](stat.set_all.md) | Set All Stats | Sets all major stats to min or max on the target character. mode='min'\|'max' (default: max). |
| [`stat.set_percent`](stat.set_percent.md) | Set Stat Percent | Sets a stat as a ratio (0.0~1.0) of its min-max range on the target character. |
