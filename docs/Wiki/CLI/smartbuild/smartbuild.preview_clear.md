# smartbuild.preview_clear

Overview
==

- Restore the snapshot captured by smartbuild.preview_wall/smartbuild.preview_floor and drop the preview state. Safe to call when no preview is active.

Parameters
==

- This command takes no parameters.

Usage
==

```js
await inzoi.cli.execute('smartbuild.preview_clear');
```
