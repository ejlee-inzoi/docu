# 📘 TraitValuePoint Data Guide

## 🧩 Struct: FB1TraitValuePointTableRow

Defines value thresholds for trait systems, used for measuring alignment or influence.

### Properties

- **Id** (`FName`): Unique identifier for the value point set.  
- **TraitValuePointInfoList** (`TArray<FB1TraitValuePointInfo>`): Ranges and min/max values associated with the trait value.
