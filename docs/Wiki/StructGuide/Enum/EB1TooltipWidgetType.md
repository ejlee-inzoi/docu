## 🎛️ Enum: EB1TooltipWidgetType

Defines the visual category of tooltip widgets shown in the UI.

- **Default**: Displays a default tooltip.  
- **Skill**: Displays information about a skill.  
- **Hope**: Displays a hope or life goal.  
- **HousingObject**: Tooltip for a placeable object.  
- **EmotionGauge**: Displays current emotional gauge.  
- **Schedule**: Tooltip for the timeline schedule.  
- **SitePreset**: Displays info about a preset site layout.  
- **HousingObjectCategory**: Shows category of the housing object.  
- **Relationship**: Shows relationship-related info.  
- **Material**: Tooltip for a material asset.  
- **MyTexture**: Displays a preview texture.  
- **CanvasAppearance**: Tooltip related to canvas appearance options.  
- **PlayerChangePortrait**: Used when changing character portrait.  
- **RelationshipProfile**: Tooltip for relationship profile view.  
- **SocialEventAction**: Displays action info from a social event.  
- **PrintedAccessoryObjectTooltip**: Tooltip for printed accessories.  
- **GameOption**: Tooltip for a game option or setting.
