## 🎛️ Enum: EB1ObjectCustomizeType

Defines how an object can be customized.

- **None**: No customization available.  
- **MaterialOnly**: Customize only the material (e.g., color or texture).  
- **Assemble**: Assemble the object from parts.  
- **Floor**: Floor-based placement customization.  
- **Printed**: Customize as a printed object.
