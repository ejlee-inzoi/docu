## ⚖️ Enum: EB1CompareType

Defines comparison operators used for conditional evaluation in gameplay logic or scripting.

- **None**: No comparison is applied.  
- **Equal**: Checks if values are equal (==).  
- **NotEqual**: Checks if values are not equal (!=).  
- **Greater**: Checks if left value is greater than right (>).  
- **Less**: Checks if left value is less than right (<).  
- **GreaterOrEqual**: Checks if left value is greater than or equal to right (>=).  
- **LessOrEqual**: Checks if left value is less than or equal to right (<=).
