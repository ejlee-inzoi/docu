## 🎛️ Enum: EB1TraitEventTriggerType

Defines which gameplay event can trigger a trait's effect.

- **None**: No trigger defined.  
- **InteractionRepeat**: Triggered when the same interaction is repeated.  
- **TrafficAccident**: Triggered by vehicle collisions.  
- **ChangeCurrency**: Triggered when money is earned or spent.  
- **CanelSchedule**: Triggered when a scheduled event is canceled.  
- **LeaveWorkOrSchoolSchedule**: Triggered upon leaving job or school.  
- **BuyNewFurniture**: Triggered by purchasing new furniture.  
- **TakeASubway**: Triggered by taking the subway or bus.  
- **NotArrivedSchedule**: Triggered when character misses a schedule.  
- **EnterNewSite**: Triggered when visiting a new site for the first time.  
- **EnteredSite**: Triggered whenever a site is entered.
