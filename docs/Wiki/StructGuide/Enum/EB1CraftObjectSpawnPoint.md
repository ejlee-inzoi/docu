## 🧷 Enum: EB1CraftObjectSpawnPoint

Specifies where a crafted object should appear once crafting is completed.

- **None**: No spawn location is defined.  
- **RightHand**: The crafted object appears in the character’s right hand.  
- **CraftingSlot**: The object is placed into the active crafting slot.  
- **Inventory**: The object is added directly to the inventory.
