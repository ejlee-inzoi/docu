## 🎛️ Enum: EB1FamilyCreateGenderControllType

Controls gender distribution rules when generating family members.

- **Random**: Select gender randomly with no constraint.  
- **Equal**: Ensure equal number of male and female members.  
- **NotEqual**: Ensure gender imbalance (not equal).  
- **Male**: Force all generated members to be male.  
- **Female**: Force all generated members to be female.  
- **Num**: Internal count marker (not used in gameplay logic).
