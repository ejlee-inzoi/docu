## 🎛️ Enum: EB1SiteState

Represents categorized options used throughout simulation and logic systems.

- **Normal**: The site is in a normal, functioning state.  
- **UnOccupied**: The site is currently unoccupied.  
- **Occupied**: The site is actively occupied.  
- **UnOccupiedEmptyLand**: The site is unoccupied and undeveloped.
