## 🔔 Enum: EB1AlertType

Represents different types of alerts or notifications triggered by character events.

- **None**: No alert is displayed.  
- **Skill**: Notification related to skill changes or progress.  
- **RelationshipChanged**: Existing relationship has been modified.  
- **RelationshipBegin**: A new relationship has been formed.  
- **RelationshipEnd**: An existing relationship has ended.  
- **NewFamilyMoveIn**: A new family has moved into the neighborhood.  
- **ValuesDisabled**: Core character values have been disabled.  
- **AmbitionDisabled**: The ambition system is currently disabled.  
- **PreferenceDisabled**: Character preferences are not available.  
- **HobbyDisabled**: Hobbies are currently disabled.  
- **JobDisabled**: The character's job system is disabled.  
- **ChangeJobId**: Job ID has changed or been reassigned.  
- **JobRankUp**: The character has been promoted.  
- **JobRankDown**: The character has been demoted.  
- **JoinCompany**: The character has joined a company.  
- **LeaveCompany**: The character has left their company.  
- **WarningFired**: A firing warning has been issued.  
- **MobileMessage**: A message has arrived via mobile interface.  
- **JobInducementType**: A job inducement or offer has occurred.  
- **PreferenceSelect**: The player is prompted to select a preference.  
- **MobileMessagePlain**: A plain mobile message has been received.  
- **LeaveCompanyByAge**: The character has retired due to age.  
- **JoinSchool**: The character has enrolled in a school.  
- **LeaveSchool**: The character has dropped out or left school.  
- **EducationRankUp**: The character's academic standing has improved.  
- **EducationRankDown**: The character's academic standing has worsened.  
- **GraduationSchool**: The character has graduated.  
- **WarningExpel**: A warning about possible expulsion has been issued.  
- **ExpelledSchool**: The character has been expelled from school.
