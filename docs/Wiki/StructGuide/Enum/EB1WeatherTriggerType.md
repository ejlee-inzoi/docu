## 🎛️ Enum: EB1WeatherTriggerType

Defines conditions that can trigger a change in weather effects or reactions.

- **None**: Indicates that no specific value is selected.  
- **SeasonChanged**: Triggered when the current season changes.  
- **WeatherChanged**: Triggered when the weather type changes (e.g., sun to rain).  
- **TemperatureRangeChanged**: Triggered when the temperature enters or exits a certain range.  
- **GoOutSide**: Triggered when a character goes outside.  
- **GoInSide**: Triggered when a character enters an indoor space.
