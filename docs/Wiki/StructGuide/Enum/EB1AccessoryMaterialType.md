## 🎨 Enum: EB1AccessoryMaterialType

Specifies the type of material used in an accessory. This type determines visual characteristics and rendering logic for accessory components.

- **None**: No specific material type.
- **Jewelry**: Represents metallic or gem-based accessories such as earrings, necklaces, etc.
- **Accessory**: General-purpose accessory material.
- **Glass**: Used for transparent or translucent items like eyeglasses.
- **Hair**: Represents hair-based accessories such as extensions or decorative strands.
