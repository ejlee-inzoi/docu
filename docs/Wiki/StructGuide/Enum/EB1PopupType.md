## 🎛️ Enum: EB1PopupType

Defines popup UI types used throughout the game.

- **Common**: General-purpose popup.  
- **CommonBuy**: Popup for purchasing.  
- **Drama**: Narrative or drama popup.  
- **Cat**: A popup featuring a Psycat.  
- **ChoiceCard**: Popup showing choice cards.  
- **SocialEvent**: Used for social event selection or result.  
- **Narration**: Narration-driven popup.  
- **None**: No popup (used as fallback or disable state).
