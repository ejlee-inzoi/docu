# 📘 CostumeEditParts Data Guide

## 🧩 Struct: FB1CostumeEditPartsTableRow

Title for each outfit part during outfit creation in character customization.

### Properties

- **Id** (`FName`): Unique Title ID for the outfit part.  
- **PartsNameTextId** (`FString`): Outfit part title displayed in the game, localized based on language settings. *(StringTableKey = "Id")*
