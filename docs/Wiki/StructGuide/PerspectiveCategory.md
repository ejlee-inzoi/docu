# 📘 Perspective Data Guide

## 🧩 Struct: FB1PerspectiveCategory

Defines a single category of perspective (e.g., belief/value) with activation settings.

### Properties

- **Id** (`FName`): Unique identifier for the perspective category.  
- **Probability** (`float`): Likelihood of this perspective being chosen or triggered.  
- **CoolTime** (`float`): Cooldown time before this perspective can be activated again.
