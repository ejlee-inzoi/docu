# 📘 Modifier Data Guide

## 🧩 Struct: FB1ModifierTableRow

Defines a modifier entry, which bundles together multiple modifier settings applied to a character or system.

### Properties

- **Id** (`FName`): Unique identifier for this modifier.  
- **ModifierList** (`TArray<FB1ModifierSettingData>`): List of individual modifier settings that define how this modifier affects stats, states, or behaviors.
