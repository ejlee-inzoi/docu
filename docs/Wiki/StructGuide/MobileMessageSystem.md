# 📘 MobileMessageSystem Data Guide

## 🧩 Struct: FB1MobileMessageSystemTableRow

Defines mobile message system entries for displaying system-level notifications or automated messages.

### Properties

- **Id** (`FName`): Unique identifier for the system message set.  
- **Systems** (`TArray<FB1MobileMessageSystemData>`): List of system message entries and their UI text properties.
