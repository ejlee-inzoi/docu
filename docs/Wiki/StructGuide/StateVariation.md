# 📘 StateVariation Data Guide

## 🧩 Struct: FB1StateVariationTableRow

Defines a group of state variations used to alter animations, appearances, or logic based on context.

### Properties

- **Id** (`FName`): Unique identifier for the variation.  
- **StateGroupInfo** (`TArray<FB1StateGroupInfo>`): List of group definitions including states.
