# 📘 CoopTalk Data Guide

## 🧩 Struct: FB1CoopTalkTableRow

Defines a cooperative talk entry, including animation, behavior, and script triggers.

### Properties

- **Id** (`FName`): Unique identifier for this coop talk entry.  
- **CoopTalkInfo** (`FB1CoopTalkInfo`): Data defining how the talk is staged, animated, and triggered.
