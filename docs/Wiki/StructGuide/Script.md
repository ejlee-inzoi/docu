# 📘 Script Data Guide

## 🧩 Struct: FB1ScriptTableRow

Defines a set of scripts grouped under a unique identifier, used in interactions, events, or conditions.

### Properties

- **Id** (`FName`): Unique identifier for this script entry.  
- **Scripts** (`TArray<FB1Script>`): List of individual scripts executed under this entry.
