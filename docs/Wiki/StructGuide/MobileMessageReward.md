# 📘 MobileMessageReward Data Guide

## 🧩 Struct: FB1MobileMessageRewardTableRow

Defines relationship point rewards for different types of message replies.

### Properties

- **Id** (`FName`): Unique relationship identifier for this reward configuration.  
- **PositivePoint** (`float`): Points granted when selecting the positive answer.  
- **NormalPoint** (`float`): Points granted when selecting the neutral answer.  
- **NegativePoint** (`float`): Points granted when selecting the negative answer.
