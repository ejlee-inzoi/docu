# 📘 Season Data Guide

## 🧩 Struct: FB1SeasonTableRow

Defines seasonal display information used for localized UI or time-based systems.

### Properties

- **Id** (`FName`): Unique identifier for the season.  
- **SeasonTextId** (`FString`): Localized label for the season (e.g., Spring, Summer). *(StringTableKey)*
