# 📘 DoorMessage Data Guide

## 🧩 Struct: FB1DoorMessageTableRow

Defines a message shown to the player when they lack permission to open a door.

### Properties

- **Id** (`FName`): Unique identifier for the door message entry.  
- **NoPermissionMessage** (`FString`): Localization key for the message shown when permission is denied. *(StringTableKey)*
