## 🧩 Struct: FB1RandomOrderInteraction

Defines an interaction entry that can be randomly selected during a scheduled order.

### Properties

- **InteractionBagId** (`FName`): ID of the interaction bag to execute. *(Foreign = "InteractionBag.Id")*  
- **Probability** (`float`): Probability (0.0–1.0) this interaction is selected.
