## 🧩 Struct: FB1AmbitionStepMissionDefine

Defines a mission within a step of an ambition, including the mission ID and optional script executed upon completion.

### Properties

- **MissionId** (`FName`): ID of the ambition-related mission to be completed (linked to `Mission_Ambition.Id`).  
- **MissionCompleteScriptId** (`FName`): Script to execute when this mission is completed (linked to `Script_Ambition.Id`).
