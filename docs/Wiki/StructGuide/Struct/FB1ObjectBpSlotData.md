## 🧩 Struct: FB1ObjectBpSlotData

Defines a slot location and configuration on a Blueprint component for object attachment or interaction.

### Properties

- **ComponentName** (`FName`): Name of the component to which the slot is assigned.  
- **SlotName** (`FName`): Identifier for the slot on the component.  
- **Location** (`FVector`): Relative location of the slot.  
- **Rotation** (`FRotator`): Relative rotation of the slot.  
- **SlotComponentData** (`FB1SlotComponentData`): Additional configuration for the slot, such as interaction, FX, object placement, or spatial logic.
