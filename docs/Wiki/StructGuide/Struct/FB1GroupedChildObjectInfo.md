## 🧩 Struct: FB1GroupedChildObjectInfo

Defines a child object that is grouped with a parent and attached via a specific slot.

### Properties

- **Id** (`FName`): ID of the child object to attach. *(Foreign = "Object.Id")*  
- **AttachSlotName** (`FName`): Name of the slot where the object is attached.
