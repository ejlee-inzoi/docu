## 🧩 Struct: FB1GrowthChangeData

Defines a growth-related data entry, including the probability of change occurring.

### Properties

- **Id** (`FName`): The ID of the data to be changed when the growth stage changes (e.g., trait, preference, etc.). 
- **Probability** (`float`): Probability that the change will happen. *(ClampMin = 0)*
