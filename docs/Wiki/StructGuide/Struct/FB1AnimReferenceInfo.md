## 🧩 Struct: FB1AnimReferenceInfo

Contains reference information for an animation, including root motion options and metadata.

### Properties

- **AnimReferenceName** (`FString`): The internal name used to reference this animation setup.  
- **bEnableRootMotion** (`bool`): If true, root motion is enabled for the animation.  
- **bForceRootLock** (`bool`): If true, the root is locked during playback.  
- **ModifiersNum** (`int32`): Number of modifiers associated with the animation.
