## 🧩 Struct: FB1DreamPoolSelection

Defines a weighted entry for selecting a dream pool from a set of options.

### Properties

- **Weight** (`float`): Probability weight used to determine selection likelihood.  
- **DreamPoolId** (`FName`): ID of the dream pool this entry refers to. *(Foreign = "DreamPool.Id")*
