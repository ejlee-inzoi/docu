## 🖼️ Struct: FB1IconWithBGInfo

Holds information for displaying an icon with two background colors, typically used in UI or item representations.

### Properties

- **IconId** (`FName`): ID reference to the icon texture asset. (Foreign: `B1Texture2D.Id`)
- **BGColor1Id** (`FName`): ID reference to the first background color mapping. (Foreign: `ColorMapping.Id`)
- **BGColor2Id** (`FName`): ID reference to the second background color mapping. (Foreign: `ColorMapping.Id`)
