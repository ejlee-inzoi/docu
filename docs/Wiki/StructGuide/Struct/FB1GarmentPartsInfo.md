## 🧩 Struct: FB1GarmentPartsInfo

Defines a single garment part and its attachment configuration.

### Properties

- **PartsId** (`FName`): Identifier for the garment part. *(Foreign = "GarmentTopParts.Id", ModTip = "Used Json (GarmentTopParts)")*  
- **PositionBone** (`FName`): Name of the skeleton bone where the part is attached.  
- **ForceVariantSuffix** (`FString`): Optional suffix used to enforce a specific visual variant.
