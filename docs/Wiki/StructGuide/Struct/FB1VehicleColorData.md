## 🧩 Struct: FB1VehicleColorData

Defines color information for a vehicle, used in UI and gameplay rendering.

### Properties

- **ColorId** (`FName`): Identifier for this color entry. *(Foreign = "ColorMapping.Id")*  
- **UIHexColor** (`FString`): Color displayed in UI as a hex string.  
- **IngameColor** (`float`): In-game color blend parameter or weight.
