## 🧩 Struct: FB1PostureDirectionInfo

Represents a posture animation assigned to a specific direction.

### Properties

- **Direction** (`EB1SlotDirection`): Defines the slot direction this override applies to (e.g., Left, Right), relative to the direction the character is facing when interacting with the slot.
- **PostureAnimId** (`FName`): Animation ID used for the posture when facing this direction.
