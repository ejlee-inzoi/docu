## 🧩 Struct: FB1TutorialContent

Defines a single tutorial sub-topic, including its title and description entries.

### Properties

- **SubTopicId** (`FName`): Unique identifier for this tutorial sub-topic.  
- **SubTitleTextId** (`FString`): Localization key for the tutorial subtitle. *(StringTableKey)*  
- **DescTextIdList** (`TArray<FB1DescriptionId>`): List of detailed description entries shown for this tutorial.
