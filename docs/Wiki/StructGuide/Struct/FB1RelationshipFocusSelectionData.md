## 🧩 Struct: FB1RelationshipFocusSelectionData

Defines automatic focus selection settings for a relationship target.

### Properties

- **IsEnable** (`bool`): If true, auto-selection is enabled for this target.  
- **AutoSelectionData** (`FB1RelationshipAutoSelectionData`): Defines the automatic selection behavior when the active avatar reaches a focus point.
Includes success and failure probabilities, and the corresponding relationship stat changes applied in each case.
