## 🧩 Struct: FB1QualityWeight

Defines weight values for each quality level based on level difference.

### Properties

- **LevelDifference** (`int32`): The gap between skill or stat level and required level.  
- **Bad** (`float`): Weight value for bad quality outcome.  
- **Fair** (`float`): Weight value for fair quality outcome.  
- **Good** (`float`): Weight value for good quality outcome.  
- **High** (`float`): Weight value for high quality outcome.  
- **Superb** (`float`): Weight value for superb quality outcome.
