## 🧩 Struct: FB1AgeAffectInfo

Defines an age-specific effect configuration used for scaling parameters or modifiers by growth step.

### Properties

- **GrowthStep** (`EB1GrowthStep`): Specifies the growth stage (e.g., Baby, Youth, Adult) where this effect applies.  
- **MiscParam1** (`int32`): Custom numeric parameter 1 used by the system for age-based calculations.  
- **MiscParam2** (`int32`): Custom numeric parameter 2 used by the system for age-based calculations.
