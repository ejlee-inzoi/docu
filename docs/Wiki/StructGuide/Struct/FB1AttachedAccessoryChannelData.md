## 🧩 Struct: FB1AttachedAccessoryChannelData

Information about customizable segments of the material.

### Properties

- **ListKey** (`FString`): Identifier for segments of the material that can be customized.  
- **AITexture** (`bool`): Whether this channel uses an AI-generated texture.  
- **DescTextId** (`FString`): Localized description for the channel, displayed in the game.
