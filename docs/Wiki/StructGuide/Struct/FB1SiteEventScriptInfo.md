## 🧩 Struct: FB1SiteEventScriptInfo

Defines a script executed by a specific role in a site event.

### Properties

- **ScriptId** (`FName`): ID of the script to execute. *(Foreign = "Script.Id")*  
- **ScriptTargetRoleId** (`FName`): ID of the target role affected by this script.
