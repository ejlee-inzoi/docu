## 🧩 Struct: FB1OverridePostureState

Specifies posture override settings, including direction-specific variants.

### Properties

- **PostureId** (`FName`): ID of the posture to override. *(Foreign = "Posture.Id")*  
- **SlotDirection** (`EB1SlotDirection`): Defines the slot direction this override applies to (e.g., Left, Right), relative to the direction the character is facing when interacting with the slot.
