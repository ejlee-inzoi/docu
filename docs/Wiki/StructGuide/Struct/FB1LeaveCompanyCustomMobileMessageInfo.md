## 🧩 Struct: FB1LeaveCompanyCustomMobileMessageInfo

Defines a custom mobile message to be sent when a character leaves a company, based on the reason for departure.

### Properties

- **Reason** (`EB1LeaveCompanyReason`): Reason the character is leaving the company.  
- **LeaveCompanyId** (`FName`): Message ID to send when the departure occurs. *(Foreign = "CustomMobileMessageTable.Id")*
