## 🧩 Struct: FB1HowlingModifierSetting

Defines a modifier used to influence howling effect weights.

### Properties

- **ModifierType** (`EB1ModifierType`): Type of modifier applied to the howling weight.  
- **ModifierKey** (`FName`): Key or identifier used to apply or look up the modifier.
