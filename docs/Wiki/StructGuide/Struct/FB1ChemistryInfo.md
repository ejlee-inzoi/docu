## 🧩 Struct: FB1ChemistryInfo

Defines the relationship chemistry value associated with a selected trait, used to evaluate compatibility between characters.

### Properties

- **TraitSelectId** (`FName`): ID of the selected trait or value, linked to `TraitSelect3.Id`.  
- **ChemistryInfo** (`EB1RelationshipChemistry`): The chemistry level associated with the selected trait (e.g., VeryGood, Bad).
