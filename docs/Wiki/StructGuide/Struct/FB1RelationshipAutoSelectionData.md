## 🧩 Struct: FB1RelationshipAutoSelectionData

Defines probability and stat effects when automatically selecting relationship targets.

### Properties

- **SuccessProb** (`float`): Probability of success during auto-selection.  
- **SuccessAddStatValue** (`float`): Stat value added if selection succeeds.  
- **FailProb** (`float`): Probability of failure during auto-selection.  
- **FailAddStatValue** (`float`): Stat value added even if selection fails.
