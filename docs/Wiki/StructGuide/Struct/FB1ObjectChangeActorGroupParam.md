## 🧩 Struct: FB1ObjectChangeActorGroupParam

Defines UI data for changing the actor group during object customization.

### Properties

- **ActorName** (`FName`): Display name of the actor group.  
- **ActorIcon** (`TSoftObjectPtr<UObject>`): Icon representing the actor group. *(AllowedClasses = Texture2D, SlateTextureAtlasInterface)*
