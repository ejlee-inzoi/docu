## 🧩 Struct: FB1RelationshipGiftObjectData

Defines an object-based gift for relationship interactions.

### Properties

- **ObjectType** (`EB1CarriableObjectType`): Type of carriable object (e.g., Item, Food, Tool).  
- **Id** (`FName`): ID of the object used as a gift. *(Foreign = "Object.Id")*  
- **Count** (`int32`): Number of objects included in the gift.
