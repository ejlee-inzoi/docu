## 🧩 Struct: FB1FloatCurve

Defines a named curve with keyframes used for value interpolation based on time.

### Properties

- **Name** (`FName`): Identifier for the curve.  
- **Keys** (`TArray<FB1FloatCurveKey>`): List of time-value key pairs that define the curve.
