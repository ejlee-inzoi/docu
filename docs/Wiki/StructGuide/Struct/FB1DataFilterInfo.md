## 🧩 Struct: FB1DataFilterInfo

Defines a data filter category used for organizing or limiting display options in menus or systems.

### Properties

- **TitleName** (`FName`): Internal name or identifier for the filter group.  
- **TitleNameTextId** (`FString`): Localization key for the display name of this filter. *(StringTableKey)*  
- **TagList** (`TArray<FB1DataFilterTagInfo>`): List of individual filter tag options within this group.
