## 🧩 Struct: FB1MobileMessagePartiesData

Defines the context and conditions for determining mobile message recipients.

### Properties

- **BaseObject** (`EB1MobileMessageBaseObject`): The base object context (e.g., Self, Target, System).  
- **Conditions** (`TArray<FB1Condition>`): List of conditions that determine whether this party receives the message.
