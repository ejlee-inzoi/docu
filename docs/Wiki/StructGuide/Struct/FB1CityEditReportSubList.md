## 🧩 Struct: FB1CityEditReportSubList

Represents a sub-entry used in city edit reports, typically for detailing individual decorative elements or parts.

### Properties

- **Id** (`FName`): Unique identifier for the report sub-entry.  
- **ListKey** (`FString`): Internal reference key for lookup or grouping.  
- **DescriptionTextId** (`FString`): Localization key used to describe this item in UI or reports.
