## 🧩 Struct: FB1AnimTrack

Defines a named animation slot and its associated sequence segments.

### Properties

- **SlotName** (`FName`): The name of the animation slot where the sequence is applied.  
- **AnimSegments** (`TArray<FB1AnimReferenceInfo>`): A list of animation references that make up the track's segments.
