## 🧩 Struct: FB1RelationshipDecayData

Defines how a relationship stat decays over time within a specific range.

### Properties

- **From** (`float`): Minimum stat value this decay rule applies to.  
- **To** (`float`): Maximum stat value this decay rule applies to.  
- **DecayValuePerSec** (`float`): Amount the stat decays per second within this range.
