## 🧩 Struct: FB1SocialEventActionCondition

Defines a condition used to determine if a social event action should be visible or available.

### Properties

- **RoleId** (`FName`): ID of the role this condition applies to.  
- **GenderType** (`EB1GenderType`): Gender filter for role participants.  
- **Count** (`int32`): Number of characters matching the condition required.
