## 🧩 Struct: FB1HowlingWeightInfo

Represents weight configuration for a howling effect, including base value and modifiers.

### Properties

- **BaseWeight** (`float`): Default weight used to calculate the likelihood of a howling effect.  
- **ModifierSettings** (`TArray<FB1HowlingModifierSetting>`): Modifiers that can dynamically adjust the base weight.
