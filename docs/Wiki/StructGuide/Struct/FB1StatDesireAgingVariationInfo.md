## 🧩 Struct: FB1StatDesireAgingVariationInfo

Defines scaling weights for desire stat behavior, applied per growth stage.

### Properties

- **Infant** (`float`): Weight applied when the character is an infant.  
- **Toddler** (`float`): Weight applied when the character is a toddler.  
- **Child** (`float`): Weight applied when the character is a child.  
- **LowTeen** (`float`): Weight applied when the character is a low-teen.  
- **HighTeen** (`float`): Weight applied when the character is a high-teen.  
- **Youth** (`float`): Weight applied when the character is a young adult.  
- **Adult** (`float`): Weight applied when the character is an adult.  
- **Old** (`float`): Weight applied when the character is old.  
- **Elderly** (`float`): Weight applied when the character is elderly.
