# 📘 StatePriority Data Guide

## 🧩 Struct: FB1StatePriorityTableRow

Defines the priority level of a state used for animation or behavior resolution.

### Properties

- **Id** (`FName`): Unique identifier for the state.  
- **StateType** (`FName`): State type name.  
- **Priority** (`int32`): Priority value for resolving state conflicts.
