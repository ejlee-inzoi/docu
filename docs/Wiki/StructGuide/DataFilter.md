## 🧩 Struct: FB1DataFilterTableRow

Defines a list of data filters used to categorize or filter items in UI or logic.

- **Id** (`FName`): Unique identifier for this filter set.  
- **FilterList** (`TArray<FB1DataFilterInfo>`): List of filter entries used for UI grouping and data.
