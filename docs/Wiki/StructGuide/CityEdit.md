# 📘 CityEdit Data Guide

## 🧩 Struct: FB1CityEditTableRow

Defines a list of city-editable assets used for customizing a city’s layout or decoration.

### Properties

- **Id** (`FName`): Unique identifier for the city edit group.  
- **CityEditList** (`TArray<FB1CityEditChangeListInfo>`): List of changeable objects or assets under this group.
