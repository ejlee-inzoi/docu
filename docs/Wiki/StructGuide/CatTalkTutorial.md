# 📘 CatTalkTutorial Data Guide

## 🧩 Struct: FB1CatTalkTutorialTableRow

Defines a tutorial entry for a chatbot, including descriptive content and structured subtopics.

### Properties

- **Content** (`TArray<FB1DescriptionId>`): Description blocks shown during the tutorial sequence.  
- **SubTopicList** (`TArray<FB1TutorialContent>`): List of subcontents of the tutorial.
