# 📘 DataSort Data Guide

## 🧩 Struct: FB1DataSortTableRow

Defines a list of sort orders used to arrange data in UI or selection systems.

### Properties

- **Id** (`FName`): Unique identifier for this sort set.  
- **OrderList** (`TArray<FB1DataSortOrderInfo>`): List of sort entries used to define order logic.
