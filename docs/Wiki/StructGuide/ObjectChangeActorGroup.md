# 📘 ObjectChangeActorGroup Data Guide

## 🧩 Struct: FB1ObjectChangeActorGroupRow

Defines a group of actors whose behavior or visuals can be changed via interaction or UI.

### Properties

- **Id** (`FName`): Unique identifier for this group.  
- **Actors** (`TArray<FB1ObjectChangeActorGroupParam>`): List of actor entries affected by group actions.
