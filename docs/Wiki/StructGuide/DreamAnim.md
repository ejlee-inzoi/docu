# 📘 DreamAnim Data Guide

## 🧩 Struct: FB1DreamAnimTableRow

Defines dream animation entries for characters during dreams.

### Properties

- **Id** (`FName`): Unique identifier for the dream animation entry.  
- **DreamAnimInfos** (`TArray<FB1DreamAnimInfo>`): List of posture-based dream animations.
